2.2. Задание 2. Обработка аргументов (args.sh)
#!/bin/bash
echo "Имя скрипта: $0"
echo "Количество аргументов: $#"
echo "Все аргументы: $@"
for arg in "$@"; do
    echo "  $arg"
done
Результат:
$ ./args.sh a b c d e f g h i j k
Имя скрипта: ./args.sh
Количество аргументов: 11
Все аргументы: a b c d e f g h i j k
Аргументы по одному:
  a b c d e f g h i j k
2.3. Задание 3. Аналог ls (myls.sh)
#!/bin/bash
DIR=${1:-.}
for file in "$DIR"/*; do
    if [ -f "$file" ]; then
        echo "Файл: $(basename "$file")"
    elif [ -d "$file" ]; then
        echo "Каталог: $(basename "$file")"
    fi
done
2.4. Задание 4. Подсчёт файлов (count.sh)
#!/bin/bash
EXT=$1
DIR=${2:-.}
COUNT=$(find "$DIR" -type f -name "*.$EXT" | wc -l)
echo "Файлов .$EXT в $DIR: $COUNT"
Результат:
$ ./count.sh sh ~/lab12
